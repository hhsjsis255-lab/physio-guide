Physio Study Guide

Clinical Concepts (Quick Revision)

Types of Pain

  • Sharp pain → acute injury
  • Dull pain → chronic condition
  • Radiating pain → nerve involvement
  • Referred pain → internal origin

Gait Patterns

  • Antalgic gait → pain avoidance
  • Trendelenburg gait → hip abductor weakness
  • Hemiplegic gait → stroke
  • Shuffling gait → Parkinson’s

Special Tests Overview

  • SLR Test → nerve involvement
  • Spurling’s Test → cervical compression
  • Lachman Test → ACL injury

Red Flags 🚨

  • Severe trauma
  • Unexplained weight loss
  • Neurological deficit
  • Loss of consciousness

Clinical Approach (Universal Steps)

  1. History taking
  2. Observation
  3. Palpation
  4. ROM assessment
  5. Special tests
  6. Clinical reasoning
